# 📊 Coffee Shop Sales Analysis

> An Excel-based data analysis project to uncover actionable insights from retail coffee shop sales data.

## 📌 Overview

This project focuses on analyzing coffee shop sales data using Microsoft Excel to identify trends, patterns, and business insights.

The goal is to support data-driven decision-making by answering key business questions related to sales performance, customer behavior, and product demand.

- Data Cleaning
- Data Visualization (Charts & Dashboards)

## 📈 Key Business Questions Answered

- How do sales vary by **day of the week** and **hour of the day**?
- What are the **peak sales hours**?
- What is the **monthly revenue trend**?
- How do sales perform across **different store locations**?
- What is the **average order value per customer**?
- Which products are **top-selling** (by quantity & revenue)?
- How do sales vary by **product category and type**?

## ✨ Key Features

- 📊 Interactive Excel dashboard
- 📅 Time-based sales analysis (hourly & daily trends)
- 🏪 Location-wise performance comparison
- 🛒 Product-level insights
- 📈 Revenue trend analysis

## 📂 Project Files

- `coffee_shop_sales_book.xlsx` → Main dataset & analysis
- `Coffee Shop Sales Analysis.pdf` → Summary report / presentation

## 🧠 Key Insights

- Identified peak sales hours to optimize staffing and operations  
- Highlighted best-selling products driving maximum revenue  
- Observed variations in sales across different locations  
- Analyzed customer purchasing patterns to improve targeting
